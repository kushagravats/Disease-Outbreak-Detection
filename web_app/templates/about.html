ğŸ”¬ Methodology

Our system employs a comprehensive multi-stage pipeline:

  1. Data Collection: Health-related tweets from Kaggle dataset (10,000+ samples from BBC, CNN, Reuters, CBC)
  2. Text Preprocessing: Cleaning, normalization, stop-word removal, and tokenization
  3. Feature Engineering: TF-IDF vectorization, sentiment analysis, keyword extraction
  4. Model Training: Naive Bayes and Logistic Regression classifiers
  5. Evaluation: Cross-validation, confusion matrices, and performance metrics
  6. Deployment: Flask web application for real-time predictions

ğŸ¤– Machine Learning Models

Naive Bayes Classifier

A probabilistic classifier based on Bayes' theorem with TF-IDF features. Achieved 85.54% accuracy with balanced precision-recall metrics.

Logistic Regression

Enhanced linear classifier with additional features including sentiment analysis. Achieved 88.55% accuracy, demonstrating superior performance.
